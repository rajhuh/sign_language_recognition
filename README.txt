# Hand Gesture Recognition

## Overview
This project implements a real-time hand gesture recognition system using computer vision and machine learning. It captures input from a webcam, processes hand landmarks, and predicts gestures through a web interface.

---

## Setup Instructions

### 1. Clone the Repository
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git  
cd YOUR_REPO  

### 2. Install Dependencies
pip install -r requirements.txt  

### 3. Run the Application
python app.py  

---

## Usage

1. Open the application in a browser:  
   http://127.0.0.1:10000  

2. Allow camera access when prompted  

3. Display hand gestures in front of the webcam  

4. The system will provide real-time predictions  

---

## Requirements
- Python 3.8 or higher  
- Webcam
